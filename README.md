# ChemaTex Website

A custom-coded static website built from the Durable AI website reference, redesigned for a more professional B2B chemical manufacturing/trading company feel.

## Files
- `index.html` — Home
- `about.html` — About Us
- `solutions.html` — Solutions
- `categories.html` — Product Categories
- `industries.html` — Industries
- `contact.html` — Contact Us
- `styles.css` — Full visual design system and responsive CSS
- `script.js` — Header/footer injection, mobile menu, WhatsApp button, and demo form behavior

## How to preview
Open `index.html` in your browser.

## How to edit
- Replace the SVG placeholder logo in `script.js` if you have the real logo file.
- Replace the visual placeholder panels with real images by editing the CSS or adding image tags.
- Connect contact forms to a backend service such as Formspree, Netlify Forms, EmailJS, or your own server.
- Replace the map placeholder in `contact.html` with a real Google Maps or Mapbox embed.

## Deployment options
You can upload this folder to Netlify, Vercel, GitHub Pages, or any basic hosting provider.


Update: Categories now include a dropdown menu with six independent category pages. Each final product has its own landing page with title, description, packaging (5 L / 20 L), and PDF preview buttons for Safety Sheet and Data Sheet placeholder files.
